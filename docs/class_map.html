Detailed Description

Classe principale gérant la logique et l'affichage du niveau.

La classe Map charge la configuration du niveau, gère les interactions entre Pacman, les fantômes et les éléments du décor (pac-gommes, murs). Elle gère également le score et l'affichage global du jeu.
