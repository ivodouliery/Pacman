\doxysubsection{Detailed Description}
Classe principale gérant le cycle de vie du jeu. 

La classe \doxylink{class_game}{Game} encapsule la fenêtre SFML, la boucle de jeu principale et délègue la logique spécifique à la carte (\doxylink{class_map}{Map}) et aux entités.
