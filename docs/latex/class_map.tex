\doxysubsection{Detailed Description}
Classe principale gérant la logique et l\textquotesingle{}affichage du niveau. 

La classe \doxylink{class_map}{Map} charge la configuration du niveau, gère les interactions entre \doxylink{class_pacman}{Pacman}, les fantômes et les éléments du décor (pac-\/gommes, murs). Elle gère également le score et l\textquotesingle{}affichage global du jeu.
