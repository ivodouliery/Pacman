\doxysubsection{Detailed Description}
Classe de base abstraite pour toutes les entités mouvantes du jeu. 

La classe \doxylink{class_entity}{Entity} fournit les fonctionnalités de base pour le mouvement, la gestion des collisions avec la grille, et l\textquotesingle{}affichage des sprites animés.
