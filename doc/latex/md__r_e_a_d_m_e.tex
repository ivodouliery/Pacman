Pacman écrit en C++ moderne (C++17) utilisant la bibliothèque SFML.\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Prérequis}{Prérequis}}\label{md__r_e_a_d_m_e_autotoc_md1}
Avant de commencer, assurez-\/vous d\textquotesingle{}avoir installé les outils suivants sur votre machine \+:


\begin{DoxyItemize}
\item {\bfseries{Compilateur C++}} compatible C++17 (GCC, Clang, MSVC).
\item {\bfseries{CMake}} (version 3.\+12 ou supérieure).
\item {\bfseries{SFML}} (version 3.\+0).
\begin{DoxyItemize}
\item {\itshape Note \+: Assurez-\/vous que SFML est correctement installé et accessible par CMake (via {\ttfamily find\+\_\+package}).}
\end{DoxyItemize}
\item {\bfseries{Catch2}} (version 3, pour les tests unitaires).
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{md__r_e_a_d_m_e_autotoc_md2}

\begin{DoxyEnumerate}
\item {\bfseries{Cloner le dépôt \+:}}

{\ttfamily bash git clone \href{https://github.com/ivodouliery/Pacman.git}{\texttt{https\+://github.\+com/ivodouliery/\+Pacman.\+git}} cd Pacman }
\item {\bfseries{Créer le dossier de build \+:}}

{\ttfamily bash mkdir build cd build }
\item {\bfseries{Configurer le projet avec CMake \+:}}

{\ttfamily bash cmake .. }
\item {\bfseries{Compiler le projet \+:}}

{\ttfamily bash cmake -\/-\/build . }
\end{DoxyEnumerate}\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysection{\texorpdfstring{Lancement du Jeu}{Lancement du Jeu}}\label{md__r_e_a_d_m_e_autotoc_md3}

\begin{DoxyItemize}

\item {\bfseries{Sur Windows \+:}} {\ttfamily cmd Pacman.\+exe }
